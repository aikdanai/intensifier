# intensifier

## How to compile sass

```
sass --watch scss:css --style compressed
```

## How to use webpack
### install module
```
npm install
```

### run webpack-dev-server
```
npm start
```


### webpack create bundle.js
```
webpack --inline --hot
```
#### or
```
./node_modules/.bin/webpack --inline --hot
```
#### or
```
./webpack-build
```

### optimize
```
webpack --inline --hot --optimize-minimize --optimize-dedupe
```
